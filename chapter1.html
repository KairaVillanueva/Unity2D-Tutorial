Programming Terms

Variables

It is a storage box. It literally lets you store any data type into a name you have created. For example, the type of data I want to store consists of the number 1. I know 1 is an integer, so I will declare it with the type int which stands for integer and I will then store this type into variable number.


How does this look like in C#?


int number = 1

This may seem abstract at first, but let us put it in terms of health points. In the game we will be creating, our spaceship has 5 health points before it vanishes, so in order for me to declare its health, I will need to store its health into a variable. Creating these variables is helpful because it saves the programmer the time to reiterate the numbers over and over again. Once a variable is created, the programmer can just reference that variable and use it throughout the script.


You can also have the type float which takes in decimal points. In C#, f is added next to the decimal number to indicate that it is of type float.


float healthPoints = 5.0f;

When naming your variables using C#, there are some important rules to be followed. These names or identifiers cannot start with a number, cannot be a C# keyword, cannot start with a capital letter, and your identifiers need to be consistent.


NOPE:

  • float 1health = 5.0f;
  • float case = 5.0f;
  • float HealthPoints = 5.0f;
